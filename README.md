# 🏛️ Census App

A modern, premium digital census data collection web application built with HTML, CSS, and Vanilla JavaScript.

## 🔐 Login Credentials

| Role | Phone | OTP |
|------|-------|-----|
| **Enumerator (User)** | Any 10-digit number (e.g. `9876543210`) | `123456` |
| **Admin** | Any 10-digit number | `654321` |

## ✨ Features

- 🌟 **Animated Splash Screen** with glowing orbs
- 🔐 **OTP-based Login** with Enumerator / Admin roles
- 🏠 **4-Step Household Wizard** — Head Info → Personal → Location → Members
- 👨‍👩‍👧‍👦 **Dynamic Member Forms** — add/remove unlimited members
- 📊 **Real-time Dashboard Stats** — households, members, today's entries
- 🔍 **Search & Filter** households by name
- 📋 **Full Detail Modal** — click any household to view all data
- 🛡️ **Admin Panel** — all households, enumerators list, reports
- 📁 **Export Data** — CSV and JSON download
- 💾 **LocalStorage Persistence** — data saved between sessions
- 📱 **Fully Responsive** — works on mobile, tablet, desktop
